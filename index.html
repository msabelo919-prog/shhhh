HOW TO PLAY

Objective

Control the snake to eat food (red squares) and grow as long as possible without hitting walls or yourself.

Controls

Keyboard:

  • Arrow Keys or WASD: Move snake
  • SPACE or P: Pause/Resume
  • R: Restart game
  • ESC: Return to menu
  • M: Toggle sound

Touch/Mouse:

  • Use directional buttons
  • Game control buttons
  • Speed adjustment buttons

Scoring

  • Each food: 10 points
  • Every 5 foods: Speed increases
  • High score is saved automatically

Tips

  • Plan your path ahead
  • Don't trap yourself in corners
  • Use the whole playing field
  • Start with Easy mode if you're new
